# SwiperJSlight Test Scaffold

This project contains an MIT-licensed Swiper Element bundle and a starter automated test setup.

## Setup

```bash
npm install
```

## Run tests

```bash
npm test
```

## Watch mode

```bash
npm run test:watch
```

## Files

- `swiperlight-element-bundle.js`: bundled Swiper Element source
- `tests/swiper-element.test.js`: starter element registration/render tests
- `tests/setup.js`: jsdom polyfills and test cleanup
- `vitest.config.js`: Vitest config (`jsdom` environment)
